# WIPE 🧹

> 스마트폰 자동 UV-C 살균 키오스크 — 1인 개발 사업 프로젝트

## 프로젝트 개요

핸드폰을 하단 투입구에 넣으면 수직 리프트로 투명 챔버에 올라가 UV-C 살균되는 무인 자판기형 키오스크.
성수동 카페/팝업 등 트렌디한 장소를 타겟으로 세균 소멸 시각화 UI로 SNS 바이럴을 핵심 전략으로 삼는다.

## 폴더 구조

```
wipe/
├── README.md
├── PROGRESS.md               ← 진행 현황 (항상 최신 유지)
│
├── docs/
│   ├── 01_business/          ← 사업 전략, 시장 분석, 수익 모델
│   ├── 02_hardware/          ← 하드웨어 설계, BOM, 구매 리스트
│   └── 03_software/          ← 소프트웨어 아키텍처, UI 설계
│
├── firmware/                 ← Arduino C/C++ 펌웨어 (추후 생성)
├── middleware/               ← Raspberry Pi Python 미들웨어 (추후 생성)
├── kiosk/                    ← Electron 키오스크 UI (추후 생성)
└── dashboard/                ← Next.js 관리자 대시보드 (추후 생성)
```

## 핵심 스펙

| 항목 | 내용 |
|---|---|
| 크기 | W380 × D280 × H1,200mm |
| 클리닝 시간 | 60초 이내 |
| 클리닝 방식 | UV-C LED (265nm) 살균 |
| 제어 | Arduino Mega (펌웨어) + Raspberry Pi 5 (UI/통신) |
| 디스플레이 | 10인치 가로형 터치스크린 |
| 결제 | 토스페이먼츠 (카드 + QR) |
| 목표 단가 | 대당 약 100만원 (프로토타입 기준) |
