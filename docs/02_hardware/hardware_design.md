# 하드웨어 구조 설계서

## 1. 기기 전체 구조

```
┌─────────────────────────────┐  ← 상단 LED 스트립 (파란 펄스)
│                             │
│  ┌─────────────────────┐    │
│  │   투명 아크릴 챔버   │    │  ← 전면/좌/우 3면 투명 아크릴 5mm
│  │                     │    │     후면: 흰색 반사판 + UV-C LED 배열
│  │   📱 폰이 서있음     │    │     사용자가 UV빛 + 폰 직접 목격
│  │   (UV-C 빛 가득)    │    │
│  │                     │    │
│  └──────────┬──────────┘    │
│             │ (리프트 축)    │
│  ┌──────────▼──────────┐    │
│  │   [핸드폰 투입구]    │    │  ← 하단 슬롯, 폰 세로로 삽입
│  └─────────────────────┘    │
│                             │
│  ┌─────────────────────┐    │
│  │   10" 가로형        │    │  ← 터치스크린 (세균 시각화 UI)
│  │   터치스크린        │    │     약 30° 기울어진 각도
│  └─────────────────────┘    │
│  [ 카드 결제 단말기 ]        │
└─────────────────────────────┘

치수: W380 × D280 × H1,200mm
```

---

## 2. 핵심 메커니즘: 수직 리프트

### 동작 시퀀스

```
[1단계] 폰 삽입
  사용자가 하단 투입구에 폰을 세로로 삽입
  → 스프링 크레들이 폰 좌우를 자동 고정
  → IR 센서가 폰 감지 확인

[2단계] 리프트 상승
  NEMA17 + T8 리드스크류가 크레들을 위로 올림
  → 폰이 투명 챔버 안으로 진입
  → 리미트 스위치로 상단 도달 확인

[3단계] 살균 시작
  UV-C LED 6개 점등
  → 사용자가 투명 패널 너머로 UV빛 속 폰 직접 목격
  → 하단 스크린: 세균 카운터 애니메이션
  → 60초 타이머 진행

[4단계] 완료
  UV-C LED OFF
  → 리프트 하강 → 폰 반환
  → 완료 화면 + QR 공유 코드
```

## 3. 수직 리프트 상세 설계

### 방식: T8 리드스크류

```
[NEMA17 스테핑 모터]
       │
[T8 리드스크류 300mm]   ← 회전 → 직선 운동 변환
       │
[리드스크류 너트] ──── [크레들 플랫폼]
                              │
                       [폰 고정 크레들]
                       (스프링 클립 방식)

가이드: 8mm 선형 레일 × 2 (좌우 각 1개)
베어링: LM8UU × 4
```

**선택 이유:**
- 정밀 위치 제어 가능
- 셀프 록킹 (역회전 방지 → 폰 무게에 안 내려옴)
- Arduino + A4988으로 제어 가능

### 폰 고정 크레들 (스프링 클립)

```
      ┌──[스프링]──┐
  ←───┤            ├───→   (스프링 장력으로 자동 클램핑)
      │   📱 폰   │
      └────────────┘
         (하단 지지대)

폰 너비 범위: 60~85mm (전 기종 대응)
하단 지지대: 두께 6mm 이상 수납
재질: 3D 프린팅 (PLA or PETG)
```

## 4. UV-C 챔버 설계

```
후면 (흰색 반사판 + UV 반사 알루미늄 테이프)
┌─────────────────────────────┐
│  💡UV-C  💡UV-C  💡UV-C    │  ← UV-C LED × 3 (상단)
│                             │
│         📱 폰               │  ← 양면 동시 조사
│                             │
│  💡UV-C  💡UV-C  💡UV-C    │  ← UV-C LED × 3 (하단)
└─────────────────────────────┘
전면 (투명 아크릴 5mm)

UV-C LED 스펙: 265~270nm, 4W, 방사각 120°
챔버 내부: 흰색 반사판 + 알루미늄 테이프 → 효율 극대화
```

> ⚠️ 안전 인터락: 리프트 상단 도달 (리미트 스위치 확인) 후에만 UV-C 점등
> UV-C는 눈/피부에 치명적. 챔버 밀폐 = 절대 조건.

## 6. 전기 블록 다이어그램

```
[12V 10A 파워서플라이]
        │
        ├──→ [Arduino Mega] ──→ [A4988] ──→ [NEMA17 리프트]
        │           │
        │           ├──→ [UV-C LED 모듈 × 6]
        │           │
        │           ├──→ [IR 센서 × 3]
        │           └──→ [리미트 스위치 × 4]
        │
        ├──→ [Raspberry Pi 5] ──→ [10" 터치스크린]
        │           │
        │           ├──→ [USB 웹캠]
        │           └──→ [Arduino (UART)]
        │
        └──→ [WS2812B LED 스트립]
```

## 7. 안전 설계 원칙

| 위험 요소 | 대책 |
|---|---|
| UV-C 노출 | 리미트 스위치 인터락 — 챔버 완전 밀폐 확인 후만 점등 |
| 폰 낙하 | 스프링 크레들 + 하단 지지대 이중 고정 |
| 과전류 | 퓨즈 + 모터 드라이버 전류 제한 |
| 비상 정지 | 비상 스위치 → Arduino 인터럽트로 즉시 UV-C 차단 |
