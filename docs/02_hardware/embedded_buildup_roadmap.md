# WIPE 임베디드 빌드업 로드맵 (직접 / 지원 / 외주)

> 작성일: 2026-06-09
> 전제: 창업자 = 프론트엔드 강점, 임베디드 입문. **두뇌(전자·펌웨어·SW)는 직접 보유**, 기구 양산만 외주.
> 안전: UV-C는 인체 유해 → **실습은 일반 LED로**, 실제 UV-C는 차폐+인터락 확실히 갖춘 뒤.

## 역할 분담 한눈에

| 영역 | 담당 |
|---|---|
| 전자 회로(브레드보드/만능기판), 펌웨어(아두이노), 미들웨어(Python), 키오스크 SW | **직접 (+나 코드/도면 지원)** |
| 프로토타입 기구 부품 (트레이·모터마운트·브라켓·센서홀더·아크릴 컷) | **내가 CadQuery/OpenSCAD → STL/STEP/DXF 생성 → 3D프린팅/레이저컷** |
| 기구 양산 외함 설계(CAD), 판금/사출 도면 | **외주** (RFP 작성 완료) |
| KC 전기안전 + EMC 전파 + 공장심사, (나중) 정식 PCB | **외주** |

## 단계별 빌드업

### Level 1 — 단일 부품 익히기 (브레드보드 · 1~2주 · 난이도 ★★☆)
- [ ] L1-1 스테핑모터: A4988 + NEMA17 회전/방향/속도 제어 — *내가 스케치 제공*
- [ ] L1-2 리미트 스위치 눌림 감지 (디바운스)
- [ ] L1-3 IR 센서로 물체 감지
- [ ] L1-4 **(UV 대신) 일반 LED를 MOSFET로 ON/OFF** (전류 스위칭 연습)
- 목표: 각 부품을 따로 다룰 줄 알기

### Level 2 — 리프트 1축 (핵심 메커니즘 · 2~3주 · 난이도 ★★★)
- [ ] NEMA17 + T8 리드스크류 + 8mm 레일 + LM8UU 조립 — *내가 모터마운트·트레이 STL 제공*
- [ ] **홈잉**(리미트 스위치를 원점으로) + 목표 위치 이동
- [ ] 트레이 플랫폼 상하 이동 시제품 (책상 위)
- 목표: "폰 트레이가 챔버까지 올라갔다 내려온다"

### Level 3 — 안전 인터락 + 살균 시퀀스 (2주 · 난이도 ★★★)
- [ ] **인터락**: 리미트로 "챔버 밀폐 확인 후에만 UV(LED) 점등" (절대 조건)
- [ ] 비상정지 → 인터럽트로 즉시 UV 차단 + 모터 정지
- [ ] **상태머신 펌웨어**: 투입 → 상승 → (밀폐확인) → UV 30초 → 하강 → 복귀 — *내가 펌웨어 제공, 본인 이해·수정*
- 목표: 안전하게 한 사이클 자동 완주

### Level 4 — 통신 + 통합 (네 강점 구간 · 2~3주 · 난이도 ★★☆)
- [ ] **RPi5 ↔ 아두이노 UART 프로토콜** 설계 (명령: START_LIFT / 상태: LIFT_DONE 등)
- [ ] **Python 미들웨어**(pyserial) + 로컬 WebSocket → 키오스크 UI의 mock을 실제로 교체
- [ ] **AI 비전**(RPi5 + 웹캠 + OpenCV): "스마트폰 형태인가?" 판별
- [ ] 결제(VAN 직가맹 단말) 연동
- 목표: UI 버튼 → 실제 하드웨어 동작

### Level 5 — 검증 (난이도 ★★★)
- [ ] **30초 살균력 실측** (LED 출력·거리 — 열린 과제) / 폰 양면 노출 방식
- [ ] **500사이클 무중단 내구** 테스트

## ⚠️ 안전 체크
- UV-C 실습은 **일반 LED 대체**, 실제 UV-C는 차폐 챔버 + 인터락 + 보호장비 후.
- 전원: 12V/5V 분리, **퓨즈**, 결선 전 멀티미터 확인. 모터 드라이버 전류 제한 설정.
