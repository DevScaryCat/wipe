## 1. 사업 개요

### 1-1. 사업 동기 및 시장 진입 배경

- **환경 변화와 보건 수요:** 지구 온난화로 인한 신종 전염병 및 팬데믹 발생 주기 단축 예상. 대중의 공중보건 및 개인 위생 인프라 수요 지속 증가.
- **정책적 확장성:** 향후 위생 관련 국가 지원 사업 확대 가능성에 대비한 초기 로컬 인프라 선점.
- **마케팅 소구점:** "스마트폰이 변기보다 오염도가 높다"는 데이터 기반의 보건 마케팅을 통해 위생 증진(명분)과 사용 빈도 증가(실리)를 동시 확보.

### 1-2. 핵심 아이디어 및 타겟 전략

- **주요 타겟:** 키오스크 및 무인 기기 사용에 익숙한 20~30대.
- **입지 선정 (공간 마케팅):**
  - **Phase 1 — 핫플레이스(성수, 홍대 등):** 핵심 타겟층의 유동인구가 집중되는 지역. 카페/팝업 내 설치.
  - **Phase 2 — 지하철 역사:** 열차 대기 시간(유휴 시간)을 활용. 단, 공공입찰 특성상 실적 데이터 확보 후 진입.
- **UI/UX 기반 바이럴:** 단순 살균을 넘어, 기기 내 디스플레이를 통해 스마트폰의 오염 부위와 살균 처리 과정을 시각적이고 직관적인 애니메이션으로 제공. 20대 사용자들의 자발적인 SNS 공유 및 입소문 유도.

### 1-3. 한계점 및 중점 해결 과제

| 과제 | 내용 | 우선순위 |
|---|---|---|
| 기술적 리스크 통제 | 기기 오작동으로 인한 스마트폰 파손 리스크 원천 차단 | 최우선 |
| 운영/유지보수 효율화 | 무인 자판기 특성상 기기 고장, 소모품 보충, 현장 관리 비용 최적화 | 높음 |
| 법적 규제/인허가 | 공공장소 설치 시 공간 점용 허가 및 관련 규제 사전 검토 | 중간 (Phase 2) |
| 리텐션 확보 | 초기 마케팅 이후에도 지속적 이용을 유도할 편의성 및 성능 입증 | 높음 |

## 2. 핵심 의사결정: 살균 전용 vs 세척+살균

### 2-1. 옵션 비교

| 항목 | 옵션 A: UV-C 살균만 | 옵션 B: 세척+살균 |
|---|---|---|
| 하드웨어 난이도 | 낮음 (LED만 제어) | 높음 (롤러/패드 기구 추가) |
| 파손 리스크 | 거의 0 (물리적 접촉 없음) | 높음 (접촉 = 스크래치, 낙하) |
| 소모품 | 없음 (UV LED 수명 1만시간) | 있음 (패드 교체 주기적 필요) |
| 유지보수 | 거의 무인 운영 가능 | 소모품 보충 인력 필요 |
| 사용자 만족도 | "보이지 않는 깨끗함" (UI로 보완) | "눈에 보이는 깨끗함" |
| 보험/법적 리스크 | 낮음 | 파손 배상 이슈 상존 |
| 추가 예산 | 없음 | +200~300만원 |

### 2-2. 결정

**→ Phase 1은 "UV-C 살균 전용"으로 확정**

- 폰에 물리적 접촉 0 → 파손 리스크 제거
- 소모품 0 → 무인 운영 가능
- 하드웨어 단순 → 개발 기간 단축
- "세균 시각화 UI"가 체감 가치를 만들어줌 → 실제로 깨끗해졌는지보다 "세균이 이렇게 많았어?"가 핵심 소구점
- 시장 검증 후 세척 기능은 모듈식으로 추가 가능

## 4. 사용자 이용 시간 (UX 타임라인)

### 시나리오 A: 살균만 (확정 방향)

| 단계 | 소요 시간 | 설명 |
|---|---|---|
| 폰 투입 | 5초 | 세로로 삽입, 스프링 크레들 자동 고정 |
| 결제 | 5초 | 카드 탭 또는 QR |
| 리프트 상승 | 3초 | 챔버까지 이동 |
| **UV-C 살균** | **30초** | **세균 시각화 애니메이션 동시 진행** |
| 리프트 하강 | 3초 | 폰 복귀 |
| 폰 회수 | 2초 | 폰 빼기 |
| 완료 화면 | 2초 | SNS 공유 QR 코드 |
| **합계** | **약 50초 ~ 1분** | |

### (참고) 시나리오 B: 세척+살균

| 단계 | 소요 시간 |
|---|---|
| 폰 투입 + 결제 | 10초 |
| 리프트 상승 | 3초 |
| 물리 세척 | 20초 |
| UV-C 살균 | 30초 |
| 건조 | 10초 |
| 리프트 하강 + 회수 | 5초 |
| 완료 화면 | 2초 |
| **합계** | **약 1분 20초 ~ 1분 30초** |

### 처리량 비교

| | A (살균만) | B (세척+살균) |
|---|---|---|
| 1회 이용 시간 | ~1분 | ~1분 30초 |
| 시간당 처리량 | ~50명 | ~35명 |
| 일 최대 처리량 (12시간) | ~600회 | ~420회 |

> 1분 이하 = 사용자 체감 "빠르다" → 재이용 의향 높음
> 2분 이상 = "기다리느니 그냥 가지" → 이탈 발생

## 6. 파손 리스크 대응 (살균 전용 기준)

UV-C 살균 전용이므로 폰에 물리적 접촉이 없어 파손 리스크가 극히 낮으나, 리프트 이동 중 사고에 대비:

| 대응 | 내용 |
|---|---|
| 하드웨어 안전 | 스프링 크레들 + 하단 지지대 이중 고정 |
| 소프트웨어 안전 | 리미트 스위치 인터락 — 챔버 밀폐 확인 후만 UV 점등 |
| 비상 정지 | 비상 스위치 → 인터럽트로 즉시 UV 차단 + 모터 정지 |
| 법적 대응 | 이용 전 면책 동의 화면 표시 |
| 보험 | 영업배상책임보험 가입 |
