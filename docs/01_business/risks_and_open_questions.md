## P0 진행 결정 (2026-06-03)

### #1 효능 표시 — 🅿️ 보류 (추후)
- "99.9% 살균" 수치 효능 주장 및 시험성적서는 **당장 추진하지 않음.**
- 마케팅은 수치 효능 대신 **"습관·재미·SNS·위생 케어"** 소구로 우회. 정식 상업화/효능 주장 시점에 시험성적서 재검토.

### #2 인증 풀스코프 — 🅿️ 보류 (추후)
- KC 전기안전 + EMC(전파) + 공장심사 합산 견적은 **추후 진행.** 파일럿(지인 매장, 비공개)까지는 인증 전 운영, 정식 배치 전 반드시 완료.

### #3 세균 카운터 정직성 — ✅ 확정 방향
**결론: 실시간 세균 측정은 불가(세균 ~1µm, 카메라로 안 보임 / ATP는 접촉식 소모품 / 배양 24h). "세균 N마리"는 본질적으로 연출 숫자.**

채택 방향 = **UV/바이올렛 형광 기반 "비포/애프터" 시각화**:
- 보라색 UV-A LED + 기존 웹캠으로 *그 폰의 실제* 지문·유분·잔여물을 형광으로 비춤 → 진짜 "비포"가 보임 (하드웨어 추가 = UV-A LED 1개로 저렴).
- 살균(UV-C) 후 깨끗해진 모습과 비교 → "애프터".
- **부위별 인지:** 형광 이미지에서 오염이 집중된 영역(카메라 주변/하단/화면 등)을 RPi5가 분석해 **"여기가 더러웠어요 → 깨끗해졌어요"를 부위별로 하이라이트** → 사용자 만족감 핵심.
- **정직선(반드시 준수):**
  - ✅ "표면 오염/유분 시각화", 평균 통계 인용("스마트폰은 평균 변기의 10~18배"), 실제 폰 사진 + 그래픽 효과 → OK
  - ⚠️ "당신의 폰: 세균 2,847마리"처럼 *그 폰을 측정한 척* = 기만광고. 금지.
  - ❌ 시험성적서 없는 "99.9% 제거" 수치 = 금지(#1과 연동).

### #4 위생 역설(공용 트레이 오염) — ✅ 해결 방향
- **구조적 자동 해결:** 트레이가 폰과 함께 매 사이클 UV-C 챔버로 진입 → **트레이 표면도 매번 자동 살균**.
- **재질:** 비다공성·UV내성 = 스테인리스 또는 아노다이즈드 알루미늄(금속이 UV 반사 → 챔버 효율 덤 상승). (PLA는 다공성+UV 열화 → 양산은 금속/항균코팅)
- **대기 중 트레이 챔버 내 '주차'/주기적 셀프 살균** → 다음 사용자가 처음 닿는 표면이 이미 깨끗.
- **UI 신뢰 전환:** "트레이는 매 사용마다 자동 살균됩니다" 표시 → 역설을 신뢰 포인트로.
- ⚠️ **열린 과제:** 폰을 눕히면 **트레이에 닿는 바닥면은 그림자 → 살균 안 됨.** 해결 옵션 = (a) 돌기/엣지 지지로 폰을 띄워 양면 노출, (b) 윗면 위주 살균 인정(가장 많이 만지는 면), (c) 석영(UV-C 투과) 트레이(고가). → "30초 살균력 실증"과 한 묶음으로 검증.

## Part 2. 현실 리스크 정리

### A. 법적 / 규제 리스크 ★가장 큼

| 리스크 | 내용 | 대응 |
|---|---|---|
| **효능 부당광고** | "99.9% 살균" 등 효능 주장은 **객관적 실증자료(시험성적서) 없으면 표시광고법 위반**. 공정위가 공기청정기 "바이러스 99.99% 제거" 부당광고 제재 전례. 칫솔살균기 15종 중 3종만 실제 99.9% 달성한 조사도 있음 → 30초 살균은 더 보수적으로 표현해야. | 항균/항바이러스 시험성적서(KTR·KCL 등) 확보 **또는** 효능 수치 직접 표기 회피("UV-C 위생 케어" 식 표현) |
| **인증 범위 누락** | 전기용품 **안전인증(KC) + 공장심사**에 더해, 모터·UV·RPi 탑재 전자기기이므로 **전자파 적합성(EMC, 국립전파연구원)** 별도 필요. 현 예산은 KC 전기 300만만 반영. | 인증 대행사에 **풀스코프 견적**(안전+EMC+공장심사) 의뢰, 예비비 확보 |
| **감독 관심 품목** | 소비자원(KCA)이 "자외선(UV) 살균제품 안전실태조사"를 수행하는 등 규제기관 모니터링 대상. | 효능·안전 표현 보수적으로 |
| **UV-C 인체 유해 / PL** | UV-C 누출은 눈·피부 손상. 사고 시 제조물책임(PL). | 인터락 이중화 + 영업배상책임보험 + 면책 고지 |
| **무인기 개인정보** | 카메라 촬영 = 개인정보. | 이미지 비저장·즉시폐기 설계 + 고지 |
| **결제/사업자** | 무인 PG/VAN 계약, 전자금융 관련 사업자 등록 필요. | 토스페이먼츠 사업자 등록 선행 |
| 공공장소 점용허가 | 지하철·옥외는 점용허가 필요 (Phase 2). | Phase 2에서 검토 (인지됨) |

### B. 경쟁 / 시장 리스크

- **"국내 경쟁사 없음"은 사실이 아님.** 개인용 폰 살균기(바디랩스·링크·넷메이트·에스엠테크원, 삼성 갤럭시 UV 무선충전기 등)는 **레드오션**. → 우리의 공백은 "제품"이 아니라 **"무인 키오스크 공용 서비스 + 입지 + 바이럴"** 모델. 마케팅·IR 메시지를 여기에 맞춰야 함.
- **진입장벽 낮음 → 카피 위험.** UV+리프트는 기술적 해자가 약함. 진짜 해자는 **선점 입지·브랜드·운영 데이터**. 빨리 깔고 브랜드를 선점하는 게 전략.
- **수요 타이밍.** 코로나 피크 대비 위생 경각심 둔화 가능 → "위생"보다 **"습관·재미·SNS"** 소구가 더 안전.
- **지불의사(WTP) 미검증.** 사람들은 그냥 물티슈로 닦음. "돈 내고 쓸까?"는 파일럿으로 검증해야 할 **최대 가설**.

### C. 운영 리스크

| 리스크 | 대응 |
|---|---|
| 무인 고장·방치 (1인 운영) | 원격 모니터링(P1-6) + 자가복구 시퀀스 |
| 트레이 잼 / 폰 끼임 → 분실·파손 클레임 | 트레이 눕힘 방식(완화됨) + 비상 회수 + 보험 |
| **위생 역설** — 공용 트레이가 오염원 | 트레이도 매 사이클 UV 노출, 사용자 인식 관리 |
| 회전율 한계 (1분/인, 1대) | 피크 시 대기 → 입지당 대수·배치 검토 |

### D. 기술 리스크

- **30초 살균력 실증** (최우선) — LED 출력·개수·조사거리로 목표 log reduction 확보되는지 실측. 안 되면 시간↑(처리량↓) 또는 LED 스펙↑(단가↑).
- 알리 UV-C LED 품질·수명·실제 파장 편차 → 양산용은 검증된 공급선 필요.
- AI 비전 이물질 오판(폴더블·케이스 오인) → 폴백 로직.
- 트레이+리프트 기구 신뢰성 — 수천 사이클 내구 테스트 필요.

### E. 창업자(1인) 역량 / 리소스 갭 — 솔직한 점검

> "만들 수 있는가"보다 **"고객이 돈 내고 쓰는가"** 검증이 비어 있는 게 가장 큰 갭.

| 갭 | 현황 / 보완책 |
|---|---|
| **전 영역 1인 부하** | HW(기구·전자·펌웨어)+SW(임베디드·프론트·백·AI비전)+사업(인증·법무·영업·마케팅) 전부는 비현실적. 기구설계 외주(진행 중)처럼 **외주/대행 적극 활용** |
| 하드웨어 양산·기구 경험 | 외주 RFP로 일부 보완. 양산성·단가 검증은 미지수 |
| 인증·법규 전문성 | **인증 대행사 + (효능표시) 법무 자문** 필요 |
| 자금 | 1,000만원으로 인증+3대+파일럿은 빠듯. EMC·시험성적서 추가 시 초과 위험 → 예비비/조달 계획 |
| 영업 채널 | 카페 입점 계약(rev-share?) 협상력·네트워크 |
| **고객 검증(Customer Discovery)** | 잠재 사용자·점주 인터뷰, WTP 테스트가 아직 없음. 제작 전 **저비용 검증(설문·간이 목업·사전예약)** 권장 |
