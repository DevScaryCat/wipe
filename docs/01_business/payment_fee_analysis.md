# 결제 수수료 분석 & 결제 경로 전략 (500원 소액결제)

> 작성일: 2026-06-09 / 근거: 금융위원회·여신금융협회 공시, 토스페이먼츠 수수료 공시 (2025.2.14 인하요율, 2026 유지)
> 검증: 리서치 5분야 병렬 조사 + 적대적 검증 후 정리

## 핵심 결론

**500원 결제는 수수료 때문에 손해 보는 구조가 아니다.** 한국은 카드 수수료가 **100% 정률(%)** 이라, 미국식 "건당 정액수수료(예: 2.9% + $0.30/건)"가 **없다.** 금액이 작으면 수수료 절대액도 비례해 작아진다. → 손익을 가르는 건 "수수료율"이 아니라 **"결제 경로 선택"**.

## 500원 1건당 실제 수수료 (경로별)

| 결제 경로 | 500원당 수수료 | 판단 |
|---|---|---|
| **VAN 직가맹 카드단말기 (영세)** | 신용 ~2원(0.4%) · 체크 ~0.75원(0.15%) | ✅ 최선 |
| 카드기반 간편결제 (영세) | ~2.8~4.2원 (카카오·네이버·토스페이) | ✅ 무방 |
| PG 웹결제 경유 | ~17~24원 (3.4~4.3% + VAT) | ⚠️ 약 10배, 회피 |
| 계좌이체 / 가상계좌 | **200원 / 400원 (정액!)** | ❌ 500원의 40~80%, 사용 금지 |

> 영세가맹점 부가가치세 **세액공제(매출의 1.3%)**까지 감안하면 실질 수수료는 0~마이너스 가능.

## 실행 전략

1. **무인 키오스크는 PG 웹결제 대신 VAN 직가맹 카드단말기(CAT/리더기)로 결제** — 마진을 가르는 결정적 선택.
2. 결제수단은 **카드 + 카드기반 간편결제**로 한정. **계좌이체·가상계좌는 열지 말 것**(정액 수수료 함정).
3. 신생 사업자는 초기 일반요율 → 영세 확인 시 **차액 소급 환급**.
4. **소액 카드결제 거부는 불법** (여신전문금융업법 제19조, 위반 시 1년 이하 징역/1천만원 이하 벌금). → "현금만"으로 회피 불가, **경로 최적화로 해결**.

## 시사점

- 500원 파일럿 가격은 **결제 수수료 관점에서 문제없음** → 가격 전략 유지 가능.
- 진짜 마진 압박 요인은 수수료(1~5원)가 **아니라** 전력·소모품·유지보수·정산 운영비, 그리고 고정비 대비 이용량.

## 주의 / 미확인

- 수치는 2025.2.14 인하요율(신용 영세 0.4%/체크 0.15%) 기준. 보수적으로 0.5%/0.25%로 잡아도 결론 불변.
- **정산수수료·입금(송금)수수료는 계약별 상이** → 위 추정엔 미포함. PG/VAN 계약 시 정산주기(D+1/D+2)·정산수수료 반드시 확인.
- 실제 가맹 계약 전 토스페이먼츠 등 현행 공시·계약서로 최종 확인 필요.
