# 사업 계획서

## 1. 사업 개요

| 항목 | 내용 |
|---|---|
| **제품명** | WIPE |
| **콘셉트** | 스마트폰 UV-C 자동 살균 무인 키오스크 |
| **핵심 차별화** | 투명 챔버로 살균 과정을 눈으로 직접 확인 + 세균 소멸 시각화 UI |

## 2. 시장 분석

### 국내 현황
- 스마트폰 보급률 95%+ (약 5천만 대)
- 스마트폰 표면 세균: 화장실 변기 대비 10~18배
- 국내 직접 경쟁 키오스크: **없음 (블루오션)**
- 해외 참고: PhoneSoap(미국 개인용), 중국 공유충전기

### 수익성 시뮬레이션

```
[편의점/카페 기준]
하루 유동인구: 500명
전환율: 2%
하루 이용: 10건 × 1,000원 = 10,000원
월 수익: 약 300,000원/대
대당 원가 100만원 기준 회수: 약 3~4개월
```

### 경쟁사 분석

| 경쟁사 | 국가 | 형태 | 우리와의 차이 |
|---|---|---|---|
| PhoneSoap | 미국 | 개인용 박스 | 무인 키오스크 아님 |
| 중국 공유충전기 | 중국 | 충전 키오스크 | 클리닝 없음 |
| 국내 | - | **없음** | 우리가 최초 |

## 3. 바이럴 마케팅 전략

### 핵심 원칙
> **기기 자체가 콘텐츠** — 사람들이 스스로 찍어서 올리도록 설계

### 바이럴 트리거
1. **투명 챔버** — UV 파란빛 속 폰이 살균되는 장면을 눈으로 직접 목격
2. **세균 카운터** — "세균 2,847마리 → 0마리" 실시간 숫자 감소
3. **공유 카드** — 완료 후 QR 스캔 → "내 폰 세균 99.9% 제거" 인스타 스토리 이미지
4. **해시태그** — 기기에 #WIPE_성수 크게 인쇄

### 파일럿 인센티브
- 인스타 스토리 공유 시 다음 이용 무료 (첫 50명)
- 인플루언서 무료 체험 협찬 (예산 70만원)
