# 최종 목표 (Final Goal)

스마트폰을 자동으로 UV-C 살균하는 무인 키오스크(자판기형)를 개발·상용화한다.
Phase 1은 "살균 전용"으로 확정. 세척 기능은 시장 검증 후 Phase 2에서 검토.
2026년 10월 MVP 프로토타입 완성, 2027년 Q1 정식 배치 목표.
