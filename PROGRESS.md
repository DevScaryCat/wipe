# 최종 목표 (Final Goal)

스마트폰을 자동으로 UV-C 살균하는 무인 키오스크(자판기형)를 개발·상용화한다.
Phase 1은 "살균 전용"으로 확정. 세척 기능은 시장 검증 후 Phase 2에서 검토.
2026년 10월 MVP 프로토타입 완성, 2027년 Q1 정식 배치 목표.

# 기억해야 할 사항 (Things to Remember)

- 1인 개발 프로젝트 (하드웨어 + 소프트웨어 전 영역)
- Phase 1 = UV-C 살균 전용 (폰 물리적 접촉 없음 → 파손 리스크 0)
- 가격: 1회 1,000원
- 사용자 이용 시간: 약 50초 ~ 1분
- 키오스크 OS: Raspberry Pi 5 (Linux/Electron)
- 펌웨어: Arduino Mega (C/C++, PlatformIO)
- 키오스크 UI: Electron + React + TypeScript
- 백엔드: Next.js + Supabase
- 핵심 차별화: 투명 챔버 + 세균 소멸 시각화 UI → SNS 바이럴
- 총 예산: 1,000만원 (KC 인증 포함)
- (추가 설계 조건) 살균수로 3% 과산화수소수 채택: 벤팅 캡 적용, 불투명 탱크 사용, 반응 금속(Fe/Cu 등) 배관 내 접촉 금지
